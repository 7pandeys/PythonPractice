## 🎯 List of Problems Catered

Here are some of the Python coding challenges covered in this project:

1. 🔢 **Count Occurrence in a String**  
   Use dictionaries or `collections.Counter` to count how many times each character appears.

2. 🔁 **Palindrome Number Using Function**  
   Check whether a number reads the same forwards and backwards using a custom function.

3. 🧱 **Create a Class with Getter and Setter**  
   Demonstrate basic object-oriented principles with encapsulation via `@property` and `@setter`.
